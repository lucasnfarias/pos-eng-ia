📌 PERGUNTA:
O que são tensores e como são representados em JavaScript?

💬 RESPOSTA:
Olá! Vamos entender o que são tensores e como eles são representados em JavaScript com TensorFlow.js.

No mundo do TensorFlow.js, tudo gira em torno de **tensores**. Pense neles como vetores ou, de forma mais familiar para nós desenvolvedores JavaScript, como listas que contêm números. Esses números são a base para os algoritmos de machine learning, que aprendem com padrões nos dados.

**Por que usar tensores?**

O TensorFlow.js não consegue trabalhar diretamente com objetos JavaScript ou tipos de dados primitivos como strings. Ele precisa de uma estrutura que contenha apenas números. É aí que entram os tensores!

**Exemplo prático:**

Imagine que você tem um array de pessoas com informações como idade, cor preferida e localização:

```javascript
const pessoas = [
  { nome: "Erick", idade: 30, cor: "azul", localizacao: "São Paulo" },
  { nome: "Ana", idade: 25, cor: "vermelho", localizacao: "Rio" },
  { nome: "Carlos", idade: 40, cor: "verde", localizacao: "Curitiba" }
];
```

Para usar esses dados em um modelo de machine learning com TensorFlow.js, você precisa transformá-los em um tensor. Uma forma de fazer isso é criar uma tabela onde cada propriedade ativa recebe um valor (por exemplo, 1) e as demais recebem zero.

No exemplo, temos 3 cores diferentes: azul, vermelho e verde. Podemos representar as preferências de cada pessoa da seguinte forma:

```
 azul vermelho verde
Erick 1 0 0
Ana 0 1 0
Carlos 0 0 1
```

Essa tabela, com apenas números, é a representação em tensor dos dados originais.

Em resumo, tensores são a forma como o TensorFlow.js lida com dados numéricos, permitindo que os algoritmos de machine learning aprendam e façam previsões. Embora a conversão para tensores possa parecer um passo extra, é essencial para que o TensorFlow.js possa processar e analisar seus dados de forma eficaz.
